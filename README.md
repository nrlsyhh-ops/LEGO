# LEGO Landing Page

This project is a responsive LEGO-inspired landing page created as part of my internship application task for WEBSE. The website is built using HTML, CSS, and JavaScript with a mobile-first approach.

## Project Overview

The landing page is designed to showcase a clean, modern, and responsive interface inspired by the LEGO website. It includes a hero section, navigation bar, product-related action buttons, content cards, footer links, social media links, and simple interactive popups.

## Features

* Responsive mobile-first layout
* Sticky navigation bar
* Mobile hamburger menu
* Search button with popup search bar
* Favourite button with empty favourite popup message
* Shopping bag button with empty bag popup message
* Animated LEGO man graphic with fade in and fade out effect
* Footer section with linked LEGO pages
* Social media footer buttons
* Smooth scroll and reveal animation effects
* Clean folder structure for HTML, CSS, JavaScript, images, and icons

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Responsive Web Design

## Responsive Design

This project uses a mobile-first design approach. The layout adjusts according to different screen sizes using CSS media queries. The navigation, hero section, cards, and footer are designed to remain readable and usable across mobile, tablet, and desktop devices.

## Interactions

The website includes simple JavaScript interactions such as:

* Opening and closing the mobile navigation menu
* Showing a search popup when the search icon is clicked
* Showing a message when the favourite icon is clicked
* Showing a message when the shopping bag icon is clicked
* Displaying reveal animations while scrolling
